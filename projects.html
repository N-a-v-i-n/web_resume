SIP (Session Initiation Protocol)

 

- Scintillate Alliance pvt Ltd

Session Initiation Protocol is a signaling protocol used for initiating, maintaining, and terminating communication sessions that include voice, video and messaging applications.
SIP is used in Internet telephony, in private IP telephone systems, as well as mobile phone calling over LTE.

Requirements

- On Start, User need to be REGISTER with a unique ID.
- Once the register completed, Initiate a dialog for establishing a call. The request is sent by a user agent client to a user agent server.
- Confirm that an entity has received a final response to an INVITE request.
- The BYE request, Signal termination of a dialog and end a call.
- CANCEL Usually means terminating a call while it is still ringing, before answer.

Language and Tools used

- Python & HTML
- Py kivy
- Asterisk PBX sever
